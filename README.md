Bharat File Manager is a modern, client-side file management web application developed using HTML, CSS, and JavaScript.  
It allows users to securely upload, preview, download, and delete files directly in the browser with a clean and responsive interface.

This project is designed for learning, practical assignments, and portfolio use.

---

## âœ¨ Features

- Client-side password lock with logout functionality
- Users can change their own password (stored locally per browser)
- Upload files with a maximum size limit of **10 MB**
- Supported file types:
  - PDF
  - DOC / DOCX
  - XLS / XLSX
  - ZIP
  - JPG / JPEG / PNG / HEIC
- Upload progress bar
- Image preview for supported image files
- PDF preview in browser
- Download and delete files
- Dark mode toggle with saved preference
- Modern card-based UI with hover effects
- Indian tricolor themed background
- Fully client-side, no backend required
- No payment or card required
