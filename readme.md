Bharat File Manager is a modern, client-side file management web application built using HTML, CSS, and JavaScript.  
It allows users to upload, preview, download, and delete files directly in the browser with a clean and responsive interface.

---

## ‚ú® Features

- Password lock with logout functionality
- Upload files with maximum size limit of **10 MB**
- File type validation:
  - PDF
  - DOC / DOCX
  - XLS / XLSX
  - ZIP
  - JPG / JPEG / PNG / HEIC
- Upload progress indicator
- Image preview for supported image files
- PDF preview in browser
- Download and delete functionality
- Dark mode toggle with saved preference
- Modern card-based UI with hover effects
- Indian tricolor themed background
- Persistent storage using browser localStorage
- No backend, no server, no payment required
